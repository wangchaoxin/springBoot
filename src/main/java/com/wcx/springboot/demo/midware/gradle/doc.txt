implementation
    The dependencies required to compile the production source of the project which are not part
    of the API exposed by the project. For example the project uses Hibernate for its internal
     persistence layer implementation.

api
    The dependencies required to compile the production source of the project which are part
    of the API exposed by the project. For example the project uses Guava and exposes public
    interfaces with Guava classes in their method signatures.

testImplementation
    The dependencies required to compile and run the test source of the project.
    For example the project decided to write test code with the test framework JUnit.
